# Ruby Sinatra Starter Application

Bluemix provides a Ruby Sinatra starter application as a template so that you can add your code and push the changes back to the Bluemix environment.


## Files

The Ruby Sinatra starter application has files as below:

*   helloWorld.rb

    This file contains Ruby code for your application written using Sinara API

*   Gemfile

    This includes all dependent Gem of your application
